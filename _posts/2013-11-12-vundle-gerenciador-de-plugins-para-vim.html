Vundle - Gerenciador de Plugins para o editor VIM

{% include shared.html %}

Salve, salve unix maniacos, segue uma dica massa.

Recentemente eu fiz uma limpa no meu .vimrc e aproveitei para instalar o Vundle para gerenciar meus plugins.

O plugin para o VIM, Vundle (ou Bundle), é um complemento muito útil para que possamos instalar e gerenciar seus plugins para o editor mais "modafuck" - o VIM.

De uma forma muito simples, você especifica em seu .vimrc quais plugins que você deseja carregar, ele se encarrega de baixar automaticamente as depedências.

Para instalar, basta clonar o repositório do projeto em seu home:

$ git clone https://github.com/gmarik/vundle.git ~/.vim/bundle/vundle

Adicionar as seguintes configurações em seu .vimrc:
/home/myuser/.vimrc

set nocompatible
filetype off

" Vundle
set rtp+=~/.vim/bundle/vundle/
call vundle#rc()

" let Vundle manage Vundle
 " required!
Bundle 'gmarik/vundle'

filetype plugin indent on     " required!
" End Vundle

Feche e salve (:wq!) o arquivo.

Instale o Vundle/Bundle da seguinte forma:

$ vim

Tecle Esc (modo de visualização dentro da tela inicial do Vim), e execute o comando abaixo:

:BundleInstall

Aguarde completar a instação e pronto, divirta-se!
