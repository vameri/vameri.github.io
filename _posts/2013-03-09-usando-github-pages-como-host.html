Usando GitHub Pages como Host para seus projetos

{% include shared.html %}

Como a ideia do blog sempre foi a de contribuir com lições aprendidas, vou compartilhar uma ótima dica que recebi por meio de um request do GitHub pelo @rafaelcastrocouto.

A dica seria a possibilidade de usar o GitHub Pages como "host" para que seja possível disponibilizar seus projetos que lá estão armazenados para o público. Para aqueles que como eu, também utilizam o GitHub Pages como hospedagem para seu site pessoal, esse recurso pode ser algo de grande valia.

Bem, e como fazer isso?

É muito simples, basta criar um branch de nome "gh-pages" no próprio site do GitHub, ou para quem preferir usando nosso bom e velho console e pronto, em algum minutos você estará disponibilizando seu projeto para o publico e ele poderá ser visto acessando sua página do GitHub seguido do nome do projeto:

http://f0rmig4.github.com/gsnake

 

Agora você pode compartilhar aquele seu experimento CSS ou aquele seu projeto "maroto" em JavaScript para a comunidade.

Então como eu sempre repito: "Que a força esteja com vocês".
