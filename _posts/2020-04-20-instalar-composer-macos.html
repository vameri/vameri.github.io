Como instalar o Composer no macOS

{% include shared.html %}

O PHP composer é uma ferramenta de gerenciamento de pacotes de dependência elegante e simples para a linguagem PHP.

Você pode instalar facilmente todos os pacotes necessários usando o composer. Ele mantém uma lista de pacotes necessários em um arquivo json chamado composer.json. Este tutorial ajuda você a instalar e configurar o composer em seu sistema operacional macOS.

1 - Pré-requisitos:

  • Terminal do macOs
  • A versão PHP 5.3 ou superior

2 - Download:

Faça o download do arquivo binário do composer no site getcomposer.org, executando o seguinte comando. Ele criará um arquivo composer.phar no diretório atual.

$ curl -sS https://getcomposer.org/installer | php

Agora, copie esse arquivo composer.phar no diretório bin para disponibilizar em qualquer parte do sistema. Além disso, defina a permissão de execução no arquivo.

3 - Configuração:

$ mv composer.phar /usr/local/bin/composer 
$ chmod +x /usr/local/bin/composer

4 - Execução:

Pronto! agora basta execute o comando composer no prompt de comando. Isso fornecerá detalhes da versão do composer, e as opções disponiveis.
